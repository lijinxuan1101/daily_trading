# 研究与发现

## 现有模版分析

### 盘前分析.md — 字段结构

| 章节 | 字段 | 类型 |
|------|------|------|
| 基本 | 日期、星期 | 文本 |
| 隔夜外盘 | 美股/港股/汇率/大宗商品涨跌幅+关键信息 | 表格 |
| 今日事件 | 宏观数据、会议、公告、行业政策 | 多选 checklist |
| 大盘研判 | 四大指数昨收 + 今日预判（趋势/支撑/压力/量能）| 混合 |
| 板块热点 | 强势板块、延续方向、回避板块、轮动判断 | 文本 |
| 持仓梳理 | 每个持仓：成本/浮盈/止损/关键价位/操作意图/触发条件 | 重复组 |
| 候选标的 | 标的/代码/理由/入场条件/仓位上限 | 表格 |
| 交易纪律 | 操作笔数上限、总仓位上限、特别提醒 | 文本 |
| 核心结论 | 1-3 句话总结 | 长文本 |

### 交易日志模版.md — 字段结构

| 章节 | 字段 | 类型 |
|------|------|------|
| 基本信息 | 日期/标的/代码/方向/账户 | 文本 |
| 交易详情 | 价格/数量/金额/手续费/成本/仓位比例/均价 | 数字 |
| 交易理由 | 买入逻辑/参考依据/风险提示 | 长文本 + checklist |
| 计划管理 | 止损位/目标一/目标二/最大亏损 | 价格+涨跌幅 |
| 纪律检查 | 7 项规则 checklist | checklist |
| 情绪心态 | 情绪状态/信心评分/市场感受 | 多选+评分+文本 |
| 复盘总结 | 卖出信息/5 个复盘问题 | 文本（平仓后填） |

## 技术方案细节

### 目录结构设计
```
daily_trading/
├── app.py                  # Flask 主程序
├── requirements.txt
├── data/
│   ├── pre_market/         # 盘前分析 JSON 文件，按日期命名
│   │   └── 2026-05-13.json
│   └── trade_logs/         # 交易日志 JSON 文件
│       └── 20260513_001.json
├── static/
│   ├── style.css
│   └── app.js
└── templates/
    └── index.html
```

### API 设计
```
GET    /api/pre-market          # 列表
POST   /api/pre-market          # 创建
GET    /api/pre-market/:date    # 详情
PUT    /api/pre-market/:date    # 更新

GET    /api/trade-logs          # 列表
POST   /api/trade-logs          # 创建
GET    /api/trade-logs/:id      # 详情
PUT    /api/trade-logs/:id      # 更新
```

### Claude 风格设计要点
- 背景：`#f9f9f7`（米白）
- 主色：`#d97706`（暖橙/琥珀色，呼应 Claude logo）
- 字体：系统字体栈，行高宽松
- 卡片：白色背景，极细边框，轻微阴影
- 圆角：`8px`
- 无多余装饰，信息密度适中
