# 任务计划

## 项目目标
构建一个 Web 应用，用于管理 A 股交易日志（盘前分析 + 交易日志），界面风格参考 Claude 设计语言。

## 需求确认
- **界面**：Web 应用（浏览器访问）
- **核心功能**：创建 + 浏览日志
- **扩展预留**：未来接入股票平台 Skills 读取行情、AI 分析功能
- **风格**：参考 Claude 的设计风格（简洁、克制、高级感）

---

## 阶段规划

### 阶段 1：技术选型与架构设计
**状态**：🔄 进行中

- [ ] 确定技术栈（前端框架、数据存储方案）
- [ ] 设计目录结构
- [ ] 确定数据模型（盘前分析 / 交易日志的字段结构）

**决策记录**：
- 数据存储：本地 JSON 文件（轻量，易于未来迁移；不依赖数据库）
- 前端：纯 HTML + CSS + Vanilla JS（无构建工具，开箱即用；便于后续扩展）
- 后端：Python Flask（轻量，本地运行，易于接入 AI / 股票 API）

## 关键决策

| 决策 | 选择 | 理由 |
|------|------|------|
| 数据存储 | JSON 文件 | 无需数据库，便于迁移，对个人使用足够 |
| 后端框架 | Python Flask | 轻量、易于接入 AI/股票 API |
| 前端方案 | 原生 HTML/CSS/JS | 零依赖，便于理解和扩展 |
| 启动方式 | 本地 `python app.py` | 简单直接 |
